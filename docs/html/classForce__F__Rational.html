Detailed Description

Force calculation for smeared fermion operators.

This class determines the force of a rational approximation for a given fermion operator. [28 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa]
