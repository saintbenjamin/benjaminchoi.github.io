Detailed Description

Force for the Wilson fermion operator with even-odd precondition.

This class calculates the force of the standard Wilson fermion. The gamma matrix representation is given as control string "Dirac"(default) or "Chiral" at the construction, which is used to construct the Fopr_Wilson instance. [19 June 2012 S.UEDA] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa]
