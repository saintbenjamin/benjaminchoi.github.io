Detailed Description

Standard even-odd preconditioned fermion action for HMC.

This class is used to define an action used in HMC. Fermion and Force operators are given at the construction. [19 Jun 2012 S.UEDA] (Coding history will be recovered from trac.) Modify this code to work. [03 Mar 2013 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa]
