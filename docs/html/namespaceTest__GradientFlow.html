Detailed Description

Test of gradientFlow. 

                       [08 Jul 2012 S.Aoki]

(Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] 4th order Runge-Kutta in commutator-free method formulated by E.Celledoni et al. FGCS 19, 341 (2003), as well as 1st and 2nd order Runge-Kutta are implemented. [10 Oct 2014 Y.Namekawa] unique_ptr is introduced to avoid memory leaks. [21 Mar 2015 Y.Namekawa] Adaptive stepsize control is implemented. [01 May 2015 Y.Namekawa]
