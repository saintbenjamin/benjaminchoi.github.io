Detailed Description

Integrator of link variable for a given conjugate momenta.

In the present implementation, exponential of matrix is determined by Taylor series, and whose degree (m_Nprec) is explicitly specified in the class definition. [25 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [03 Mar 2013 Y.Namekawa]
