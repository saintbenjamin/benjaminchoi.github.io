Detailed Description

Two-point correlator for staggered fermions.

This class implements the staggered hadron correlators with non-local sink operators. For mesons, quark and antiquark are contracted at different points within spatial 2x2x2 cube. No phase factor is multiplied for present implementation. As for baryons, only the nucleon correlator is implemented so far, while Delta-baryon is possible to constructed. Cf. Gupta et al., Phys. Rev. D 43, 2003 (1991). [16 Oct 2020 H.Matsufuru]

  • PS meson correlator is written in the first version. [28 Dec 2011 H.M.]
  • Baryon correlator is added. [13 Aug 2018 H.M.]
