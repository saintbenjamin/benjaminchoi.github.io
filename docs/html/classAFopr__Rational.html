Detailed Description

template<typename AFIELD>
class AFopr_Rational< AFIELD >

Fermion operator for rational approximation.

This class generates fermion operator with a rational approximation for a given fermion operator (given to the constructer). Shift-solver is used which is at present set to the CG solver explicitly. [05 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa]
