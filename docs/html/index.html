Lattice QCD common code development Project

[Introduction]

Bridge++ is a code set for performing calculations in lattice QCD on linux workstations, and
supercomputers using "C/C++" standard language with MPI.

[Environment]

  • Target platforms are listed in Makefile.
  • For confirmation on each environment, see Bridge++ wiki. https://www.bridge-hpc.org/

[Install]

  • See INSTALL.txt in the code set.
