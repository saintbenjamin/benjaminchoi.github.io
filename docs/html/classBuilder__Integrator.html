Detailed Description

Builder of MD integrator for HMC.

This class builds a molecular dynamics integrator for hybrid Monte Carlo update algorithm by composing subclasses of Integrator. At present standard leapfrog and omelyan integrators can be composed of. [25 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [03 Mar 2013 Y.Namekawa]
