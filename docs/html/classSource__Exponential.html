Detailed Description

Exponentially smeared source for 4-spinor fermion.

This class sets an exponentially smeared source vector for the 4-spinor (Wilson-type) fermion. [19 Feb 2012 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] Add set_all_color_spin,etc [ 4 Apr 2017 Y.Namekawa]
