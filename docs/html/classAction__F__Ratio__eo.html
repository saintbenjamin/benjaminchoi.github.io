Detailed Description

HMC action for Hasenbusch preconditioned fermions.

This class is used to define an fermion action used in HMC which is given as a ratio of two fermion operators. Two sets of fermion and Force operators are given at the construction. [05 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) Even-odd is implemented. [03 Mar 2013 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa] change the action to H_prec (DdagD)^{-1} Hdag_prec from Hdag_prec (DdagD)^{-1} H_prec [ 7 Jul 2019 I.Kanamori]
