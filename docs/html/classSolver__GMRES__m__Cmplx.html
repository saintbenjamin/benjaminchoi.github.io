Detailed Description

GMRES(m) algorithm with complex variables.

This class implements GMRES(m) algorithm for nonhermitian matrix. The product of vectors is treated in complex. See Y.Saad and M.H.Schultz, SIAM J.Sci.Stat.Comput. 7 (1986) 856. [8 Aug 2012 Y.Namekawa] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] Multi-threaded. [17 Jul 2014 Y.Namekawa] Introduce unique_ptr to avoid memory leaks. [21 Mar 2015 Y.Namekawa] Add restart. [22 Feb 2016 Y.Namekawa] Add flop_count. [ 8 Aug 2016 Y.Namekawa] Add use_init_guess. [ 7 Jul 2017 Y.Namekawa] Fix a bug of h[ij], which is kindly reported by Ji-Chong Yang. [ 7 Mar 2019 Y.Namekawa]
