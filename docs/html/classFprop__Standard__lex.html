Detailed Description

Get quark propagator for Fopr with lexical site index.

This is temporary implementation. [28 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) Redundant argument of fopr is deleted. [03 Mar 2013 Y.Namekawa] Introduce unique_ptr to avoid memory leaks. [21 Mar 2015 Y.Namekawa] Add flop_count. [ 8 Aug 2016 Y.Namekawa]
