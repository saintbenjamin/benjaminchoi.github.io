Detailed Description

Test of HMC update for clover fermions.

Test of spectroscopy with clover fermion.

This class tests HMC update for dynamical clover fermions. Smearing of gauge configuration for the fermion operator is incorporated. This test class obtains the quark propagator for clover fermion and calculates typical hadron correlators. The quantum numbers of hadrons are specified with gamma matrices (GammaMatrix class instance) whose set is defined in a subclass of GammaMatrixSet class.
