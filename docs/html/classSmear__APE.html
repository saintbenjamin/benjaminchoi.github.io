Detailed Description

APE type smearing of link variables.

This class smears link variables with APE-type construction of smeared links with a given projection operator to SU(N) group element. Parameter is \rho(\mu,\nu), which in general depends on the directions of the modified link and the staple. By explicitly giving \rho(\mu,\nu) as std::vector object, anisotropic setup is possible, while isotropic setup requires only one double parameter, ‘rho_uniform’. [08 Apr 2012/15 Jul 2012 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa]
