Detailed Description

HMC force class for rectangular gauge action.

Gauge action with plaquette and rectangular Wilson loops. Iwasaki, Luscher-Weisz, DBW2 are examples of this type of action. [28 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa]
