Detailed Description

APE type smearing of spatial link variables.

This class smears spatial link variables with APE-type construction of smeared links with a given projection operator to SU(N) group element. Parameter is \rho, which specifies the mixing rate of original thin link and staples. [09 Aug 2012 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa]
