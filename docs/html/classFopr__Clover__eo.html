Detailed Description

Even-odd Clover fermion operator.

This class is an even-odd version of Clover fermion operator. At present this is rough implementation, while correctly works, and to be updated by supplying complete functionality. Only the functions needed for even-odd preconditioned solver are ready. [20 June 2012 S.UEDA] (Coding history will be recovered from trac.) Modify this code to work. [03 Mar 2013 Y.Namekawa] Multi-threaded. [12 Jul 2014 H.Matsufuru] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa] Modifed toward ver.2.0. [04 Dec 2021 H.Matsufuru]
