Detailed Description

Action for staggered fermion: temporary implementation.

This is Action class dedicated to staggered fermion in even-odd site index, as a temporary implementation. It may be not well organized and to be improved. [28 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa]
