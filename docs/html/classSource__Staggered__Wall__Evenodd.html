Detailed Description

Setting source vector with wall source for staggered fermion.

This class set the wall source for the staggered fermion spectroscopy. On give time slice, the source vector is set to 1 for all spacial sites (even source) or (-1)^{x+y+z} (odd source). Gauge fixing is needed before the measurement. Cf. R.Gupta et al., Phys. Rev. D 43, 2003 (1991). [28 Dec 2011, updated 15 Oct 2020 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] Renamed: Source_Staggered_Wall -> Source_Staggered_Wall_Evenodd. Implementation is also slightly modified. [12 Oct 2020 H.Matsufuru] Modified so that odd values of Nx, Ny, Nz are allowed. [14 Jan 2023 H.Matsufuru]
