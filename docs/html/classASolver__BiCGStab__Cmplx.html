Detailed Description

template<typename AFIELD>
class ASolver_BiCGStab_Cmplx< AFIELD >

This class implements BiCGStab algorithm for nonhermitian matrix. The product of vectors is treated in complex. [12 Feb 2012 Y.Namekawa] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] Multi-threaded. [10 Jul 2014 H.Matsufuru] Introduce unique_ptr to avoid memory leaks. [21 Mar 2015 Y.Namekawa] Add restart. [22 Feb 2016 Y.Namekawa] Add flop_count. [ 8 Aug 2016 Y.Namekawa] Add use_init_guess. [ 7 Jul 2017 Y.Namekawa] Add a prescription to improve stability of BiCGStab, recommended by Kanamori-san. See G.L.G.Sleijpen and H.A.van der Vorst, Numerical Algorithms 10(1995)203-22. [26 Apr 2018 Y.Namekawa] Class template version. [27 Jul 2019 H.Matsufuru] Add initial guess, coeff_t [ 4 Jun 2020 I.Kanamori]
