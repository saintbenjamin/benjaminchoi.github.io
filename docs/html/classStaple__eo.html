Detailed Description

Staple construction.

Staple construction for even-odd gauge field. [28 Dec 2011 H.Matsufuru] Factory is introduced. [24 Jan 2017 Y.Namekawa] Multi-threading is applied. [02 Dec 2021 H.Matsufuru]
