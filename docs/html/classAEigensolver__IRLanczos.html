Detailed Description

template<typename FIELD, typename FOPR>
class AEigensolver_IRLanczos< FIELD, FOPR >

Eigenvalue solver with Implicitly Restarted Lanczos algorithm.

This class determines eigenvalues and eigenvectors for a given fermion operator. Low- or high-lying eigenmodes are determined by chaning SortField class object. [28 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa] Refactored as a template class. [29 May 2018 H.Matsufuru]
