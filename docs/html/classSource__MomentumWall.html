Detailed Description

Momentum wall source for 4-spinor fermion.

This class sets a momentum wall source vector for the 4-spinor (Wilson-type) fermion. [29 Jan 2014 N.Ishii] In order to commit to Bridge++ main stream, comments etc... are modified. [19 Feb 2014 S.Ueda] Add set_all_color_spin,etc [ 4 Apr 2017 Y.Namekawa]
