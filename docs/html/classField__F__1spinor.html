Detailed Description

Staggered-type fermion field.

This class defines 1-spinor fermion field, which is mainly used by staggered-type fermions. [28 Dec 2011 H.Matsufuru]

Added two functions mult_Field_Gn() and mult_Field_Gd() intended to be used for fermion gradient flow equation since the covariant derivative has no spin index. [25 May 2017 Y.Taniguchi]
