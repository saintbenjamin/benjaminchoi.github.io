Detailed Description

APE type smearing of link variables.

This class is alternative to the Smear_APE class. [08 Apr 2012 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa]
