Detailed Description

Force calculation for clover term of clover fermion.

This class calculates contribution clover term to the clover fermion action. [28 Dec 2011 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa]
