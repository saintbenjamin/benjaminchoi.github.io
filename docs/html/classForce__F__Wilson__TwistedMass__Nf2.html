Detailed Description

Force for the twisted-mass Wilson fermion operator.

This class calculates the force of the twisted-mass Wilson fermion. The gamma matrix representation is given as control string "Dirac"(default) or "Chiral" at the construction, which is used to construct the Fopr_Wilson instance. [23 Dec 2011 H.Matusfuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa]
