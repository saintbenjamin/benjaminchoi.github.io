Detailed Description

Wall source for SF boundary propagator.

This class sets a wall source vector for the 4-spinor (Wilson-type) fermion. [12 Apr 2012 Y.Taniguchi] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa]
