Detailed Description

Clover fermion operator.

This class implements the clover (improved Wilson) fermion operator. Wilson kernel and clover term are implemented in other classes, and this class holds them as objects. (The implementation was modified after revision 645: before that, clover term was being implemented inside this class.) The ‘mode’ controls which of D, Ddag, H, DdagD are multiplied when mult or mult_dag is called. [first ver. 24 Dec 2011/ modified 28 Aug 2012 H.Matsufuru] (Coding history will be recovered from trac.) YAML is implemented. [14 Nov 2012 Y.Namekawa] unique_ptr is introduced to avoid memory leaks [21 Mar 2015 Y.Namekawa] Modified toward ver.2.0. [04 Dec 2021 H.Matsufuru]
