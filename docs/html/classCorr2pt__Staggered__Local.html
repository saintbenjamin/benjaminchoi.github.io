Detailed Description

Two-point correlator for staggered fermions.

This class implements the local staggered hadron correlators. The local correlators mean that the sink operator is composed of spatially local contraction of quark propagators. For meson, this implies that the PS, SC, VT, AV correlators (this notation follows Appendix of Bowler et al., Nucl. Phys. B296 (1988) 732) can be described. For baryon, only nucleon correlator can be locally composed. [15 Oct 2020 H.Matsufuru] Modified so that odd values of Nx, Ny, Nz are allowed. [14 Jan 2023 H.Matsufuru]
